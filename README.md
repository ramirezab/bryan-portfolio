# Bryan Ramírez Ampié — Portfolio

Personal portfolio site for **Bryan Ramírez Ampié**, Electrical Engineer & Developer
specializing in fire alarm system design, engineering documentation, and AI-assisted
AutoCAD automation.

🔗 **Live site:** https://ramirezab.github.io/bryan-portfolio/

## About this site

A lightweight, dependency-free static site (HTML + CSS + vanilla JS), hosted on GitHub Pages.

```
bryan-portfolio/
├── index.html        # Single-page portfolio
├── styles.css        # Styling
├── script.js         # Nav, scroll reveal, image lightbox
└── assets/work/      # Anonymized engineering work samples
```

## Privacy

All engineering drawings shown are **anonymized excerpts** of real work. Client names,
project addresses, company title blocks, drawing numbers, and any other identifying
information have been removed to protect the privacy of all parties. Only the technical
drawing content is shown. Original source documents are intentionally **not** included in
this repository (see `.gitignore`).

## Local preview

It's a static site — just open `index.html` in a browser, or serve the folder:

```bash
python -m http.server 8000
# then visit http://localhost:8000
```
